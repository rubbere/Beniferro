# Beniferro Pool (Home Assistant)

Local integration for Beniferro Gen2 & Gen3 pool controllers
https://docs.myswimmingpool.eu/beniferro/local-api

## Features
- Fully local API (no cloud)
- Automatic sensor detection
- pH, ORP, temperature, pressure sensors
- Flow and tank level monitoring
- Relay status

## Installation via HACS
1. Add this repo as custom repository
2. Install integration
3. Restart Home Assistant
4. Add integration via UI

## Configuration
Enter the IP of your Beniferro device.

## Notes
This integration is read-only (API limitation).
